

# Aniket Pandey — Personal Portfolio

> 🚀 A modern, responsive personal portfolio website built with pure HTML, CSS & JavaScript — featuring smooth animations, dark/light mode, and a clean developer aesthetic.

---

## 🌐 Live Demo

[View Portfolio →]( https://portfolio-aniket-pandey.vercel.app)


## ✨ Features

- 🎨 Dark / Light mode toggle
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎭 AOS scroll animations
- 🧲 Floating social media bar
- 📊 Animated skill progress bars with IntersectionObserver
- 🗂️ Project cards with hover overlay
- 🏆 Timeline-based achievements section
- 📬 Contact form with basic validation
- ⬆️ Scroll to top button
- ⚡ Preloader animation

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure & Semantics |
| CSS3 | Styling, Animations, Responsive Design |
| Vanilla JavaScript | Interactivity & DOM Manipulation |
| AOS Library | Scroll Animations |
| Font Awesome 6 | Icons |

## 🚀 Getting Started

### Clone the repo

```bash
git clone https://github.com/aniket-pandey-ji/portfolio.git
cd portfolio
```

### Run locally

No build tools needed. Just open `index.html` in your browser:

```bash
# Option 1 — Direct open
open index.html

# Option 2 — VS Code Live Server (recommended)
# Right-click index.html → Open with Live Server
```
